## ✨Features
- Animated ticker banner
- Sticky navigation bar
- Hero section  with live dashboard mocckup
- Trusted by / logos section
- Features grid with hover effects
- How it works section
- Testimonials section
- Pricing tiers (Free, Pro, Enterprise)
- Call-to-action banner
- Custom colorful cursor
- Scroll reveal animations
- Fully responsive design

## 🧑‍💻Built with 
-HTML5
-CSS3 (Custom properties,Flexbox, Grid, Animations)
-Vanilla JavaScript

## 🚀Getting Started 
1. Clone the repo git clone (https://github.com/Badboyvado/Dappy-Landing-Page/)
2. Open index.html in your  browser No installations or dependencies needed
